# Test Plan – Wikipedia Search Functionality

## 1. Objective

The objective of this testing is to verify that the search functionality on Wikipedia works correctly, returns relevant results, and handles edge cases and invalid input properly.

## 2. Scope

This test covers the search feature available on the Wikipedia homepage.

## 3. Tested Feature

* Search input field
* Search results page
* Handling of user input

## 4. Test Types

* Functional testing
* Exploratory testing
* Negative testing

## 5. Test Environment

* Application: Wikipedia (web)
* Browser: Brave
* OS: Ubuntu 16.04

## 6. Entry Criteria

* Application is accessible via web browser
* Search functionality is available

## 7. Exit Criteria

* All planned test cases are executed
* Test results are documented
* Bugs (if found) are reported

## 8. Risks

* Limited access to backend behavior
* Testing is performed only on one browser
